# 2차 프로젝트 2팀 회의록

- 날짜: 2026년 3월 23일
- 참여자: 윤상이 (팀장), 현길용, 김희성

## 오전 회의 내용

- 백엔드 전 기능 최종 점검을 위한 테스트 시나리오 검토 및 예외 상황 리스트 작성
- 프론트엔드 최종 연동 전 데이터 포맷 및 API 권한 설정 재확인

## 작업 목표

- 백엔드 최종 점검: 사용자 관리, AI 채팅, 가상 결제, 관리자 통계 등 전 도메인 로직 전수 조사
- 시스템 안정화: 예외 처리 로직(GlobalExceptionHandler)의 정상 작동 여부 및 런타임 에러 포착 확인
- 데이터 정합성 검증: 결제 승인 후 플랜 승급 및 토큰 충전 로직의 데이터 일치성 확인

## 작업 과정

- 도메인 통합 테스트: 회원가입부터 포트원 V2 결제, PRO 플랜 승급, AI 채팅 토큰 차감까지 이어지는 전체 프로세스 반복 검증
- 결제 검증 로직 확정: 백엔드 사후 검증 결과가 실제 DB 매출 데이터 및 관리자 통계에 정확히 반영되는지 확인 완료
- 관리자 기능 점검: 플랜별 유저 분포 및 AI 모델별 사용량 집계 쿼리의 정확성 최종 테스트

## 작업 회고

- 코드 완성도 향상: 최종 점검 과정에서 미세한 비즈니스 로직의 빈틈을 발견하고 보완하여 시스템 전체의 안정성을 확보함
- 협업 효율화: 백엔드 파트의 안정적인 마무리를 통해 프론트엔드 팀이 UI 연동 및 디자인 마감에 집중할 수 있는 환경을 제공함

## 내일 목표

- 프론트엔드 최종 연동: 백엔드 API와 UI 간의 전체 연결 상태 점검 및 실배포 환경 테스트
- 발표 자료 준비: 프로젝트 핵심 기능 시연 및 사용 기술 스택 정리 작업 착수
