# 2차 프로젝트 2팀 회의록

- 날짜: 2026년 3월 12일
- 참여자: 윤상이 (팀장), 현길용, 김희성

## 오전 회의 내용

- 앨런 AI API 통신을 위한 AlanAiClient 구현 및 실시간 토큰 차감 기능 테스트
- 채팅 세션 생성 및 과거 대화 내역 조회 API 개발 완료
- 로그인, 회원가입등 user 관련 부분 프로트 작성

## 작업 목표

- 사용자 기능 완성: 내 정보 조회, 수정, 탈퇴 API 구현 및 서비스 로직 개발
- 인증 시스템 최적화: CustomUserDetails에 UUID를 포함하여 토큰 기반 인증 처리 고도화
- 예외 처리 통합: 글로벌 핸들러를 통한 404(조회 실패), 400(로직 오류) 응답 규격화

## 작업 과정

- UserEntity 내부에 updateProfile, withdraw 등 도메인 주도 설계를 위한 비즈니스 메서드 추가
- JwtAuthenticationFilter 및 UserMapper 수정: 인자 개수 불일치 에러 해결 및 ID 기반 인증 객체 생성 로직 적용
- UserService 단위 테스트 수행: 회원가입 시 첫 가입자 ADMIN 부여 및 중복 체크 로직 검증 완료


## 작업 회고

- JPA Dirty Checking 활용: 별도의 save 호출 없이 엔티티 상태 변경만으로 DB 업데이트를 처리하여 코드 가독성 향상
- 식별자 일원화: 이메일 대신 UUID를 주요 식별자로 활용함으로써 보안성을 강화하고 API 확장성을 확보함

## 내일 목표

- 앨런 AI API 연동: RestTemplate을 활용한 실시간 AI 질문/답변 통신 모듈 구현
- 채팅 도메인 고도화: ChatSession 및 MessageEntity를 활용한 대화 이력 저장 및 통계 데이터 연동
