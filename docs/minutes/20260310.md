# 2차 프로젝트 2팀 회의록

- 날짜: 2026년 3월 10일
- 참여자: 윤상이 (팀장), 현길용, 김희성

## 오전 회의 내용

- 작성하던 부분 마무리 및 비밀번호 찾기 로직 작성 및 팀원과의 협업할 부분 수정및 적용
- 프론트 메인화면 하던 작업 이어서 진행

## 작업 목표

- 사용자 관리 API 구현: 내 정보 조회, 수정(유저네임/비밀번호), 회원 탈퇴(Soft Delete) 기능 완성
- 메인페이지 작성 완

## 작업 과정

- UserEntity 고도화: updateProfile, withdraw, decreaseTokens 등 도메인 주도 설계를 위한 비즈니스 메서드 추가
- Security & JWT 디버깅: JwtAuthenticationFilter에서 UserMapper 호출 시 인자 개수 불일치 에러 해결 (ID, Username, Role 3개 인자 적용)
- Service 테스트 코드 작성: 첫 가입자 ADMIN 부여 로직 및 중복 유저네임 체크 로직 검증 완료


## 작업 회고

- Java 기반의 JPA Dirty Checking을 활용해 별도의 save 호출 없이 정보 수정을 구현하며 코드의 간결함을 확보함
- JWT 구조 변경 시 관련 필터와 매퍼를 동시에 수정해야 하는 의존성 문제를 해결하며 Security 흐름을 명확히 파악함

## 내일 목표

- 앨런 AI API 연동: RestTemplate 또는 WebClient를 활용하여 실제 AI 질문/답변 기능 구현
- 채팅 도메인 개발: ChatSession 및 ChatMessage 엔티티 설계와 토큰 실시간 차감 로직 연동
- 메인페이지 마무리 작업
