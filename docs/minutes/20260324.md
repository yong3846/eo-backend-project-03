# 2차 프로젝트 2팀 회의록

- 날짜: 2026년 3월 24일
- 참여자: 윤상이 (팀장), 현길용, 김희성

## 오전 회의 내용

- 프론트엔드 및 백엔드 실배포 환경 연동 테스트 진행 상황 공유
- 프로젝트 발표 자료 구성 및 시연 시나리오 논의

## 작업 목표

- 프론트엔드 최종 연동: 백엔드 API와 UI 간의 전체 연결 상태 점검 및 실배포 환경 테스트
- 발표 자료 준비: 프로젝트 핵심 기능 시연 및 사용 기술 스택 정리 작업 착수
- 오류 수정: AI 이용 시 이전 질문 및 답변을 인식하지 못하는 문제점 금일 내 처리 완료

## 작업 과정

- 프론트엔드 연동: 실배포 환경에서 UI와 백엔드 API 간의 통신 확인 및 데이터 흐름 점검
- 발표 준비: 팀원별 사용 기술 스택 정리 및 주요 기능(AI 채팅, 가상 결제 등) 시연 자료 작성 착수
- 디버깅: AI 채팅 기능 테스트 중 이전 대화 내역(Context) 누락 이슈 확인 및 원인 파악

## 작업 회고

- 실배포 환경에서 프론트엔드와 백엔드 API의 최종 연결 상태를 점검하며 전체적인 시스템 안정성을 확인함
- AI 대화 문맥 단절이라는 핵심 기능의 오류를 사전에 발견하였으며, 즉각적인 원인 파악과 함께 금일 내로 수정하여 서비스 완성도를 높이고자 함
- 발표 자료와 시연 준비를 본격적으로 시작하여 프로젝트 마무리에 박차를 가함

## 내일 목표

- AI 대화 문맥 유지 오류 수정 완료 확인 및 최종 QA 진행
- 팀 발표 자료(PPT 및 기술 스택 정리 문서) 내용 최종 보완
